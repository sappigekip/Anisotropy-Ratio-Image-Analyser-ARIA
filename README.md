# Anisotropy & Ratio Image Analyser (AR_IA)

ARIA is a FIJI plugin that allows you to create anisotropic and ratiometric images from dual-channel microscopy data.
Additional visualisation and analysis tools are also available such as:
  -Intensity and hue modulated display, to transform your images into easily readable and colourfull visualisations
  -Automated ROI detection with the help of Cellpose, allowing cells and structures to be identified and isolated with ease
  -Pixel value extraction from ROI's, with batch processing options
  -Median filtering also availalbe
  -Photo-bleaching/activation tracker that will change ROI size and position based on the nature and movement of your activated region (experimental)
  
How to install:
  Dowload the AR_IA.jar file and paste it in your Fiji app in the plugins directory with its other jar friends.
  Restart Fiji and the tool will be available in your plugin tab under the name AR IA.
